# clihist

Move over gnuplot, there's a new sheriff in town.

## Installation

Using go get:

```
$ go get github.com/wernerandrew/clihist
```

To just make the binary, no funny stuff:

```
$ git clone git@github.com:wernerandrew/clihist.git
$ cd clihist
$ go build
```
